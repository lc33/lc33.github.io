Problem

Users of 1Password do not use the password generator feature in order to strengthen passwords on their existing accounts or when creating new accounts.

Prototype 2

User Testing by RITE Method

RITE differs from a “traditional” usability test by emphasizing extremely rapid changes and verification of the effectiveness of these changes. Basically, RITE classifies issues into four categories based on difficults and time on fixing the issue. For example, category 1 issue means it could be fixed immediately, and test it with the next participant. Category 2 issue needs more time and will be tested as soon as it being fixed and so on.

Final Design

To instill trust, we determined that a user-centered, easy to use program would be the foundation. So our final design foucus on three points:

1

Redesigning the interface using icons that users are familiar with.

2

Integrating the password generator into the browser.

3

Allowing users to see how secure their password is compared to 1Password’s suggestion.
